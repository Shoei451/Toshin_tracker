Features

4つの主要機能

受講管理に必要なことをシンプルにまとめた。複雑な設定は不要。

📊
ダッシュボード

全体の完了率・今月と来月の予定コマ数・直近の東進模試をひと目で確認できる。月ごとのフィルタリングにも対応。

📖
講座別進捗

コマ単位で完了チェックと受講予定日を設定できる。予定日を設定するとカレンダーアプリに自動同期。月ごとの目標コマ数も設定可能。

⚡
高速基礎マスター

英単語・英熟語・英文法などのステージ進捗を管理。各ステージに「取り組む月」を設定して月別の目標を立てられる。

📅
カレンダー

受講予定日と東進模試をカレンダー形式で表示。月間の予定コマ数・完了コマ数・模試数をサイドバーで確認できる。

Screens

各画面の紹介

実際のUIのイメージ。シンプルで操作しやすい設計を意識した。

Courses

講座別進捗管理

サイドバーで講座を選択し、コマ単位でのチェック・予定日設定を行う。コマをタップするとモーダルが開き、完了チェックと受講予定日を同時に設定できる。

  • コマグリッドで直感的に進捗を視覚化
  • 予定日を設定するとカレンダーに自動追加
  • 月ごとの目標コマ数と達成率を表示
  • 講座の追加・削除に対応

⚠️
著作権・免責事項について

このアプリは個人が独自に開発したツールであり、東進ハイスクール・ナガセ等とは一切関係がありません。アプリ内のデータ（進捗・目標・スケジュール等）はすべてユーザー自身が入力するものであり、東進の教材・コンテンツそのものは一切含みません。

Tech Stack

技術スタック

フレームワークなし。素のHTML / CSS / JSとSupabaseだけで動いている。

Frontend
Vanilla JS (ES Modules)
HTML + CSS + JavaScript のみ。フレームワーク不使用。
Database & Auth
Supabase
PostgreSQL + 認証 + Row Level Security で安全に管理。
Hosting
Netlify
静的ファイルをドロップ / Githubからインポートするだけでデプロイ。ビルド不要。
Calendar Integration
calendar_app
同一Supabaseプロジェクト内の別アプリと受講予定・模試を共有。
Design
CSS Custom Properties
CSS変数でトークン管理。レスポンシブ対応済み。
Auth
Email + Password
Supabase Auth でユーザーごとのデータを分離管理。
